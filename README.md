# 五十音図ゲーム（五十音图游戏集）

这是一个帮助学习日语假名的游戏集合网站。目前包含以下游戏：

## 游戏列表

### 1. 听音选字
通过听力练习来学习日语假名的发音。玩家需要听到发音后，从选项中选择正确的假名。
